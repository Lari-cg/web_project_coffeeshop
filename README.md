# Triple Espresso
Nombre del proyecto: Web_project_cofeeshop
Descripción del proyecto: Página web de una cafeteria con videos, formularios, uso de links, entre otras herramientas. Para mostrar información, recetas y poder reservar una mesa.
Planes de mejora: Añadir un menú de lo que ofrece la cafetería y una sección para comprar merch relacionada con el café (tazas, café en bolsa, étc.)
